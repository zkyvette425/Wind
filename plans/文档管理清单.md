## 🔒 强制使用机制

### 1. 触发点检查清单
```markdown
□ 会话开始 → 检查 CLAUDE.md + 当前TODO + 最新变更
□ 标记"已测试" → 检查 测试验证管理.md + 运行真实测试  
□ 发现质量问题 → 检查 质量事故档案.md + 记录新问题
□ 技术决策 → 检查 纲领.md + 技术研究记录.md + 决策记录.md
□ 代码修改 → 更新 版本变更日志.md + TODO状态
```

### 2. 使用验证机制
- **强制引用**: 在关键决策时必须引用相关文档条目
- **使用记录**: 在文档中记录实际使用时间和场景
- **效果跟踪**: 定期检查文档使用是否真正解决了问题
